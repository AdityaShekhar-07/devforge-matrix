# DevForge Matrix

DevForge Matrix is a collection of high-performance, open-source utility tools designed for developers, creators, and professionals. 

**[🚀 Visit Live Demo](https://devforge-matrix.vercel.app/)**

## The Toolset
*   **Dev Utilities**: Markdown Previewer, URL Encoder/Decoder, CSV to JSON Converter.
*   **Media Tools**: Meme Generator, Image to WebP Converter, Image Text Extractor (OCR).
*   **Security & Crypto**: Password Generator, SHA-256 Hash Generator.
*   **Office Utilities**: Lorem Ipsum Generator.
*   **Growth Tools**: LinkedIn Post Formatter.

## Tech Stack
*   **Framework**: React
*   **Styling**: Tailwind CSS
*   **Processing**: Native Web APIs (Canvas, Crypto, etc.)
